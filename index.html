▶ SETUP GUIDE — HOST ON GITHUB PAGES
01

FORK OR CLONE THIS REPO

Go to GitHub.com → New Repository → Name it retro-arcade (or anything you like). Upload all the files from this project to your repo.

02

ENABLE GITHUB PAGES

In your repo: Settings → Pages → Source → Deploy from branch → select main branch → / (root) → Save. Your site goes live at https://yourusername.github.io/retro-arcade

03

ADD YOUR OWN ROMS (OPTIONAL)

To embed games directly (only ROMs you legally own), add them to a /roms/ folder and update the game cards in index.html to point to the ROM path. The EmulatorJS player will load them automatically.

  • NES: /roms/nes/mygame.nes
  • SNES: /roms/snes/mygame.smc
  • GBA: /roms/gba/mygame.gba
04

LEGAL FREE GAME SOURCES

  • archive.org — huge legal library with browser play built in
  • itch.io homebrew NES ROMs — original games, freely released
  • NESWorld Homebrew — public domain and freeware NES games
05

EMULATORJS CDN SETUP

This site uses EmulatorJS via CDN. For individual game pages, create a file like play.html using the template in /player/index.html. No server needed — it runs 100% in the browser.
